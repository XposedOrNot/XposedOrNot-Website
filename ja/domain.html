ドメインの検証と検証


ドメインでのデータ侵害の危険範囲を評価するには、次の手順に従ってください。3つの簡単なステップあなたの情報を確認するため。

私たちは、ドメイン ユーザーの安全を保つことが非常に重要であることを理解しています。そのため、ドメインの承認を検証するための 3 つの素晴らしい方法があります。DNS、HTML、 そしてEメール。私たちの目標は、お客様がニーズに合った最適な方法を簡単に選択できるようにすることです。

DNS: 最初の方法では、ドメインの DNS 設定に特別なレコードを追加します。これは、ドメインに秘密のハンドシェイクを行って、それが本物であることを確認するようなものです。プロセスをご案内します。プロセスが完了すると、すぐにドメインの ID が検証されます。

HTML: より実践的なアプローチをご希望の場合は、対応させていただきます。独自のコードを含む HTML ファイルを Web サイトのルート ディレクトリに追加するだけです。それは、「はい、これは私のドメインです!」というデジタル旗を立てるようなものです。フラグを確認します。フラグがあれば問題ありません。

Eメール: 最後に、電子メールの利便性を好む方のために、確認リンクを受信箱に直接送信できます。リンクをクリックすると、ドメインが認証されます。お気に入りのメールを開いてボタンをタップするだけで簡単です。

覚えておいてください、私たちはこのプロセスをできるだけスムーズかつフレンドリーにするためにここにいます。したがって、自分にとって最適な方法を選択して、そのドメインを承認しましょう。 🎉
