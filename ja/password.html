公開パスワードとは何ですか?

へようこそ公開されたパスワードは、一般の人々がパスワードが漏洩してデータ侵害によって漏洩していないかどうかを確認できるように設計された無料のプラットフォームです。

当社の膨大なパスワード コレクションは、世界中のさまざまなデータ侵害で流出した実際のパスワードで構成されています。この情報の安全性を確保するために、当社では、と呼ばれる安全性の高いハッシュ アルゴリズムを使用しています。 SHA-3 (ケチャック-512) 検証のためにパスワードを一方向ハッシュでハッシュ化して保存します。パスワードは平文で保存されません。

プラットフォームのセキュリティを強化するために当社が導入している技術的および運用上の管理の詳細については、ブログ投稿をご覧ください。「Xposed Passwords: 無料のパスワード チェッカー」。

package main

import (
	"crypto/sha3"
	"fmt"
	"io/ioutil"
	"net/http"
	"net/url"
	"strings"
)

func main() {
	var password string
	fmt.Println("Enter the password:")
	fmt.Scanln(&password)

	if len(password) != 0 {
		hash := sha3.New512()
		hash.Write([]byte(password))
		pwd_hash := hash.Sum(nil)[:10]
		apiUrl := "https://passwords.xposedornot.com/api/v1/pass/anon/" + url.QueryEscape(string(pwd_hash))

		resp, err := http.Get(apiUrl)
		if err != nil {
			fmt.Println(err)
			return
		}
		defer resp.Body.Close()

		if resp.StatusCode != 200 {
			fmt.Println("Oops: Password looks exposed and hence avoid using this")
		} else {
			fmt.Println("Password is safe")
		}
	} else {
		fmt.Println("Oops! Try again with a valid password.")
	}
}
                     
extern crate tiny_keccak;
extern crate reqwest;
extern crate hex;

use std::io;
use tiny_keccak::Keccak;
use std::str;

fn main() {
    let mut password = String::new();
    println!("Enter the password:");
    io::stdin().read_line(&mut password).unwrap();

    if !password.trim().is_empty() {
        let mut hasher = Keccak::new_keccak512();
        hasher.update(password.as_bytes());
        let mut res: [u8; 64] = [0; 64];
        hasher.finalize(&mut res);
        let pwd_hash = hex::encode(&res[0..5]); // Get 10 characters of hash
        let apiUrl = format!("https://passwords.xposedornot.com/api/v1/pass/anon/{}", urlencoding::encode(&pwd_hash));

        let resp = reqwest::get(&apiUrl).unwrap();
        if !resp.status().is_success() {
            println!("Oops: Password looks exposed and hence avoid using this");
        } else {
            println!("Password is safe");
        }
    } else {
        println!("Oops! Try again with a valid password.");
    }
}

パスワードが漏洩した場合はどうすればよいですか?

1.すぐにパスワードを変更してください: データ侵害でパスワードが侵害されたことが判明した場合は、他の場所で使用していない強力でユニークなパスワードにすぐに変更してください。

2.二要素認証をオンにする: 2 要素認証を有効にして、アカウントにセキュリティ層を追加します。これにより、攻撃者がアクセスすることが困難になります。

3.アカウントアクティビティを確認してください: アカウントのアクティビティ ログを調べて、誰かがログインしようとしたり、設定を変更しようとしたりするなど、奇妙なアクティビティや不正なアクティビティがないかどうかを確認します。

4.秘密の質問を更新する: アカウントにセキュリティの質問が関連付けられている場合は、セキュリティの質問も侵害されている可能性があるため、更新してください。

5.他のアカウントを確認してください: 他のアカウントに同じまたは類似のパスワードを使用していないか確認し、強力でユニークなパスワードに更新してください。

6.財務状況に注意してください: 銀行口座とクレジット カード口座に、身に覚えのない取引や承認されていない取引がないか確認してください。

7。違反を報告する: 侵害されたアカウントを所有する企業であるか政府機関であるかに関係なく、データ侵害について関係当局に知らせてください。

8.パスワードマネージャーを使用する: すべてのアカウントに対して強力で一意のパスワードを作成して保存するには、パスワード マネージャーの使用を検討してください。

9.警戒を怠らない: 個人情報の盗難や詐欺行為の兆候に注意し、疑わしいものはすぐに報告してください。

10.オンライン セキュリティについて詳しく見る: オンライン セキュリティのベスト プラクティスについて学び、最新の脅威と脆弱性に関する情報を常に入手してください。
