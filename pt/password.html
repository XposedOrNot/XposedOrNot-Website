Diretrizes de segurança de senha

O que são senhas Xposed?

Bem-vindo aoSenhas Xposed, uma plataforma gratuita projetada para ajudar o público em geral a verificar se uma senha foi comprometida e exposta em uma violação de dados.

Nossa enorme coleção de senhas é composta por senhas reais que foram expostas em diversas violações de dados em todo o mundo. Para garantir a segurança dessas informações, usamos um algoritmo de hashing altamente seguro chamado SHA-3 (Keccak-512) para hash e armazenar as senhas em um hash unidirecional para verificação.Nenhuma senha é armazenada em texto simples.

Para obter mais informações sobre os controles técnicos e operacionais que implementamos para aumentar a segurança de nossa plataforma, confira nossa postagem no blog'Senhas Xposed: seu verificador de senhas gratuito'.

package main

import (
	"crypto/sha3"
	"fmt"
	"io/ioutil"
	"net/http"
	"net/url"
	"strings"
)

func main() {
	var password string
	fmt.Println("Enter the password:")
	fmt.Scanln(&password)

	if len(password) != 0 {
		hash := sha3.New512()
		hash.Write([]byte(password))
		pwd_hash := hash.Sum(nil)[:10]
		apiUrl := "https://passwords.xposedornot.com/api/v1/pass/anon/" + url.QueryEscape(string(pwd_hash))

		resp, err := http.Get(apiUrl)
		if err != nil {
			fmt.Println(err)
			return
		}
		defer resp.Body.Close()

		if resp.StatusCode != 200 {
			fmt.Println("Oops: Password looks exposed and hence avoid using this")
		} else {
			fmt.Println("Password is safe")
		}
	} else {
		fmt.Println("Oops! Try again with a valid password.")
	}
}
                     
extern crate tiny_keccak;
extern crate reqwest;
extern crate hex;

use std::io;
use tiny_keccak::Keccak;
use std::str;

fn main() {
    let mut password = String::new();
    println!("Enter the password:");
    io::stdin().read_line(&mut password).unwrap();

    if !password.trim().is_empty() {
        let mut hasher = Keccak::new_keccak512();
        hasher.update(password.as_bytes());
        let mut res: [u8; 64] = [0; 64];
        hasher.finalize(&mut res);
        let pwd_hash = hex::encode(&res[0..5]); // Get 10 characters of hash
        let apiUrl = format!("https://passwords.xposedornot.com/api/v1/pass/anon/{}", urlencoding::encode(&pwd_hash));

        let resp = reqwest::get(&apiUrl).unwrap();
        if !resp.status().is_success() {
            println!("Oops: Password looks exposed and hence avoid using this");
        } else {
            println!("Password is safe");
        }
    } else {
        println!("Oops! Try again with a valid password.");
    }
}

O que devo fazer se minhas senhas forem expostas?

1.Mude sua senha imediatamente: se você descobrir que sua senha foi comprometida em uma violação de dados, altere-a imediatamente para uma senha forte e exclusiva que você não usa em nenhum outro lugar.

2.Ative a autenticação de dois fatores: adicione uma camada extra de segurança à sua conta ativando a autenticação de dois fatores, o que dificulta o acesso de invasores.

3.Verifique a atividade da sua conta: consulte os registros de atividades da sua conta para ver se houve alguma atividade estranha ou não autorizada, como alguém tentando fazer login ou alterar suas configurações.

4.Atualize suas perguntas de segurança: se você tiver dúvidas de segurança associadas à sua conta, atualize-as, pois elas também podem ter sido comprometidas.

5.Verifique suas outras contas: verifique se você usou senhas iguais ou semelhantes para outras contas e atualize-as para senhas fortes e exclusivas.

6.Fique de olho nas suas finanças: verifique suas contas bancárias e de cartão de crédito em busca de transações que você não reconhece ou não autorizou.

7.Relate a violação: Informe as autoridades relevantes sobre a violação de dados, seja a empresa proprietária da conta comprometida ou uma agência governamental.

8.Use um gerenciador de senhas: considere usar um gerenciador de senhas para criar e armazenar senhas fortes e exclusivas para todas as suas contas.

9.Fique alerta: fique atento a quaisquer sinais de roubo de identidade ou atividade fraudulenta e denuncie imediatamente qualquer coisa suspeita.

10.Saiba mais sobre segurança on-line: aprenda sobre as melhores práticas de segurança online e mantenha-se informado sobre as ameaças e vulnerabilidades mais recentes.
