Domain-Verifizierung & Validierung

Verifizieren Sie Ihren Domain-Besitz und erhalten Sie Zugriff auf die Überwachung von Datenlecks in drei einfachen Schritten.

Domain-Verifizierung

Um Ihren Domain-Besitz zu verifizieren und Zugriff auf die Überwachung von Datenlecks zu erhalten, befolgen Sie diese einfachen Schritte:

  1. Geben Sie unten Ihren Domainnamen ein
  2. Wählen Sie Ihre bevorzugte Verifizierungsmethode:
    • DNS-Verifizierung – Sicherste Methode (10 Min.) DNS-Verifizierungsanleitung (wird in neuem Tab geöffnet)
    • HTML-Verifizierung – Alternative Methode (15 Min.) HTML-Verifizierungsanleitung (wird in neuem Tab geöffnet)
  3. Folgen Sie der geführten Anleitung für Ihre gewählte Methode

Wählen Sie unten Ihre bevorzugte Verifizierungsmethode


DNS-TXT-Verifizierung
Fortgeschritten

Erstellen Sie wie unten gezeigt einen temporären TXT-Eintrag in Ihrem DNS. Anleitung ansehen (wird in neuem Tab geöffnet)

Wichtige Hinweise:
  • Lassen Sie diese Seite während des gesamten Verifizierungsprozesses geöffnet
  • Die Verifizierungsschlüssel ändern sich bei jedem Versuch – verwenden Sie stets den zuletzt angezeigten Schlüssel
  • Warten Sie auf die DNS-Propagierung (kann 10–30 Minuten dauern)
  • Nutzen Sie MX Toolbox (wird in neuem Tab geöffnet), um zu bestätigen, dass Ihr TXT-Eintrag sichtbar ist, bevor Sie auf „Verifizieren“ klicken
Validierung per META-Tag! Bitte erstellen Sie wie unten gezeigt einen temporären META-Tag auf Ihrer Startseite.


HTML-Datei-Verifizierung
Einfacher Upload

Laden Sie eine Verifizierungsdatei in das Stammverzeichnis Ihrer Website hoch. Anleitung ansehen (wird in neuem Tab geöffnet)

Einfache Schritte:
  • Erstellen Sie eine neue Datei mit dem unten angezeigten exakten Namen
  • Kopieren Sie den Verifizierungscode exakt wie angezeigt
  • Laden Sie diese Datei in das Stammverzeichnis Ihrer Website hoch
  • Die Datei sollte unter folgender Adresse erreichbar sein: https://yourdomain.com/[filename-below]
Ihre Verifizierungs-URL: Verifizierungs-URL öffnen (wird in neuem Tab geöffnet)
E-Mail-Verifizierung
Einfachste Methode
Wichtig: Verifizierungs-E-Mails werden nur an E-Mail-Adressen gesendet, die in den DNS-Einträgen Ihrer Domain aufgeführt sind (wie admin@example.com). Wählen Sie diese Methode nur, wenn Sie Zugriff auf diese E-Mail-Konten haben.
Die Validierung per E-Mail ist eine einfache Möglichkeit, eine Domain zu validieren, sofern Sie E-Mails an den autoritativen E-Mail-Kontakten empfangen können. Bitte wählen Sie die E-Mail aus und klicken Sie dann auf Verifizieren.
