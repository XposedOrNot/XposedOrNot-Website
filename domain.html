Domain Verification & Validation

Verify your domain ownership and access breach monitoring with three simple steps.

Domain Verification

To verify your domain ownership and access breach monitoring, follow these simple steps:

  1. Enter your domain name below
  2. Choose your preferred verification method:
    • DNS Verification - Most secure method (10 mins) DNS verification guide (opens in new tab)
    • HTML Verification - Alternative method (15 mins) HTML verification guide (opens in new tab)
  3. Follow the guided instructions for your selected method

Choose your preferred verification method below


DNS TXT Verification
Advanced

Create a temporary TXT record in your DNS as shown below. View guide (opens in new tab)

Important Notes:
  • Keep this page open during the entire verification process
  • Verification keys change with each attempt - always use the latest key shown
  • Wait for DNS propagation (can take 10-30 minutes)
  • Use MX Toolbox (opens in new tab) to confirm your TXT record is visible before clicking verify
META tag based validation.! Please create a temporary META data tag in your home page as shown below.


HTML File Verification

Simple Steps:
  • Create a new file with the exact name shown below
  • Copy the verification code exactly as shown
  • Upload this file to your website's root directory
  • The file should be accessible at: https://yourdomain.com/[filename-below]

Email Verification
Easiest Method
Important: Verification emails will only be sent to email addresses listed in your domain's DNS records (like admin@yourdomain.com). Choose this method only if you have access to these email accounts.
Email based validation is easy to validate a domain provided, you can receive emails in the authoritative email contacts. Please select the email and then click Verify.
