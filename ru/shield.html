Privacy Shield

Скройте свой email из публичного поиска по утечкам. Ваши данные остаются конфиденциальными. Только вы видите собственные результаты.

Активировать Privacy Shield

Укажите свой email, чтобы исключить его из публичных результатов поиска на XposedOrNot. Вы по-прежнему сможете проверять собственные раскрытые данные — просто их не увидят посторонние.

Укажите свой адрес email, чтобы активировать Privacy Shield и скрыть его из публичного поиска по утечкам.
Как это работает: после отправки проверьте почту — для завершения активации перейдите по ссылке из письма-подтверждения.
Подождите: слишком много попыток. Повторите через несколько минут.

Как это работает

Шаг 1

Укажите свой email

Отправьте email, который хотите скрыть из публичного поиска.

Шаг 2

Подтвердите владение

Перейдите по ссылке подтверждения, отправленной на вашу почту.

Шаг 3

Вы скрыты

Ваш email не будет появляться в публичном поиске по утечкам.
