இலவச தரவு மீறல் API

எங்கள் இலவச RESTful API மூலம் கோடிக்கணக்கான வெளிப்பட்ட பதிவுகளைச் சரிபார்க்கவும். API விசை தேவையில்லை.

அங்கீகாரம்

பெரும்பாலான எண்ட்பாயிண்ட்கள் பொதுவானவை. டொமைன் எண்ட்பாயிண்ட்களுக்கு ஒரு API விசை தேவை.

விகித வரம்பு

ஒரு IP-க்கு வினாடிக்கு 2 கோரிக்கைகள்; கூடுதலாக ஒவ்வொரு எண்ட்பாயிண்டுக்கும் மணிநேர மற்றும் தினசரி வரம்புகள். வரம்புகளைப் பார்க்க.

பதில் வடிவம்

அனைத்து பதில்களும் நிலையான HTTP குறியீடுகளுடன் JSON ஐ வழங்குகின்றன.

api.xposedornot.com

அதிகாரப்பூர்வ SDK-கள்

சில வரிகள் குறியீட்டில் மட்டுமே XposedOrNot ஐ உங்கள் பயன்பாடுகளில் ஒருங்கிணைக்க எங்கள் அதிகாரப்பூர்வ நூலகங்களைப் பயன்படுத்தவும்.

Node.js / JavaScript

npm மூலம் நிறுவவும்: 

npm install xposedornot

விரைவு எடுத்துக்காட்டு:

const { XposedOrNot } = require('xposedornot');

const xon = new XposedOrNot();

// Check email for breaches
const result = await xon.checkEmail('test@example.com');
console.log(result);

API விரைவு குறிப்பு கடைசியாக புதுப்பிக்கப்பட்டது: 03-Jun-2026

XposedOrNot மீது நீங்கள் காட்டிய ஆர்வத்திற்கு நன்றி. எங்கள் API ஐ முடிந்தவரை அணுகக்கூடியதாகவும் விரைவாக பதிலளிக்கக்கூடியதாகவும் வைத்திருந்து, அதை அனைவருக்கும் ஒரு மதிப்புமிக்க கருவியாக்குவதே இலக்கு. நேரடியாக Google Infrastructure இல் ஹோஸ்ட் செய்யப்பட்டு, Cloudflare ஆல் வலுவாக கேஷ் செய்யப்பட்ட XposedOrNot API, கோரிக்கைகளின் தோற்ற இடம் எதுவாயினும் அவற்றுக்கு விரைவான பதில்களை வழங்குகிறது.

XposedOrNot API, REST கட்டமைப்பின் கொள்கைகளைப் பின்பற்றுகிறது. இது JSON-குறியாக்க பதில்களை வழங்குகிறது மற்றும் நிலையான HTTP பதில் குறியீடுகளைப் பயன்படுத்துகிறது. எங்கள் பெரும்பாலான API வழிகளுக்கு அங்கீகார தேவைகள் எதுவும் இல்லை, இதனால் எளிதான அணுகலும் பயன்பாட்டின் எளிமையும் உறுதிசெய்யப்படுகின்றன. இருப்பினும், டொமைன் தொடர்பான தரவை வினவப் பயன்படுத்தப்படும் எங்கள் குறிப்பிட்ட API வழிக்கு API விசை அங்கீகாரம் தேவை என்பதை தயவுசெய்து கவனத்தில் கொள்ளவும். குறிப்பிட்ட டொமைன்களின் சரிபார்க்கப்பட்ட உரிமையாளர்கள் அல்லது அதிகாரிகள் மட்டுமே இந்த உணர்திறன் தகவலை அணுக முடியும் என்பதை உறுதிசெய்யவே இந்த நடவடிக்கை உள்ளது.

எண்ட்பாயிண்ட்களைப் பரிசோதிக்க, XposedOrNot API ஐ ஊடாடும் வகையில் ஆராயக்கூடிய API Playground ஐ முயற்சிக்கவும்.

எடுத்துக்காட்டுகளைப் படித்தல்: {curly braces} இல் காட்டப்படும் மதிப்புகள் இடம்பிடிப்பான்கள். ஒவ்வொன்றையும் (அடைப்புக்குறிகள் உட்பட) உங்கள் சொந்த மதிப்பால் மாற்றவும். எடுத்துக்காட்டாக, {email} என்பது test@example.com ஆக மாறுகிறது.

வேக வரம்புகள்

இலவச நிலையின் வரம்புகள், IP முகவரி மற்றும் எண்ட்பாயிண்ட் வாரியாக:

எண்ட்பாயிண்ட் வாரியான இலவச நிலை வரம்புகள்
எண்ட்பாயிண்ட்வினாடிக்குமணிநேரத்திற்குநாளுக்கு
/v1/check-email/{email}225100
/v1/breach-analytics225100
/v1/breaches250100
/v1/domain-breaches22550
/v1/metrics5/min50100

HTTP 429 என்பது மணிநேர அல்லது தினசரி வரம்பை அடைந்துவிட்டீர்கள் என்பதைக் குறிக்கிறது: மீண்டும் முயற்சிக்கும் முன் சாளரம் மீட்டமையும் வரை காத்திருங்கள். பதிலில் உயர் வரம்பு திட்டங்களுக்கான இணைப்பு உள்ளது.

அதிக வேக வரம்புகள் தேவையா?

இலவச API தனிப்பட்ட மற்றும் குறைந்த அளவு பயன்பாட்டுக்கானது; அது இலவசமாகவே இருக்கும். மீறல் தரவின் மீது ஒரு தயாரிப்பை உருவாக்குகிறீர்கள் அல்லது அதிக செயல்திறன் தேவைப்பட்டால், xonAPI+ மாதம் $5 முதல் கட்டண திட்டங்களை வழங்குகிறது; நிமிடத்திற்கு 25,000 கோரிக்கைகள் வரை வரம்புகள், API-விசை அணுகல், வணிக ஆதரவு ஆகியவை அடங்கும். அதே மீறல் தரவுதான்; இதுவே இலவச நிலையை இலவசமாக வைத்திருக்கிறது.

GET 1. API அழைப்பு: மின்னஞ்சல் முகவரி தரவு மீறல்களைச் சரிபார்த்தல்

ஒரு மின்னஞ்சல் முகவரி அறியப்பட்ட எந்தவொரு தரவு மீறலிலும் சம்பந்தப்பட்டுள்ளதா என்பதை இந்த API சரிபார்க்கிறது. இது மீறல்களின் விரிவான தரவுத்தளத்தைத் தேடி, மின்னஞ்சல் ஆபத்தில் இருந்தால் உங்களை எச்சரிக்கிறது.


API எண்ட்பாயிண்ட்: 
https://api.xposedornot.com/v1/check-email/{email}

details வினவல் இல்லை boolean பதிலில் விரிவான மீறல் தகவலைச் சேர்க்கவும். இயல்பாக false (true/false, yes/no, 1/0 ஐ ஏற்கிறது).
வெற்றிகரமான மீறல் கண்டறிதலின் எடுத்துக்காட்டு: ஒரு மீறல் கண்டறியப்படும்போது, இது போன்ற ஒரு JSON பதிலை நீங்கள் பெறுவீர்கள்:

GET 2. API அழைப்பு: மின்னஞ்சல் முகவரிகளுக்கான தரவு மீறல் பகுப்பாய்வு

எங்கள் API ஒரு மின்னஞ்சல் முகவரியின் தரவு மீறல் வரலாற்றின் ஆழமான பகுப்பாய்வை வழங்குகிறது. மீறல்கள் எப்போது, எங்கு நிகழ்ந்தன என்பதை இது வெளிப்படுத்தி, இந்த சம்பவங்களின் தாக்கத்தையும் தீவிரத்தையும் மதிப்பிட அத்தியாவசிய பகுப்பாய்வுகளை வழங்குகிறது. தரவு வெளிப்பாட்டு அளவுகளைப் புரிந்துகொள்வதற்கும் பாதுகாப்பு உத்திகளை மேம்படுத்துவதற்கும் இந்தக் கருவி முக்கியமானது.

API இரண்டு சாத்தியமான விளைவுகளுடன் பதிலளிக்கிறது: வெற்றி அல்லது தோல்வி. வெற்றிகரமான பதிலின் முக்கிய கூறுகள் கீழே கொடுக்கப்பட்டுள்ளன:

ஒவ்வொரு API கோரிக்கைக்கும் வினாடிக்கு 2 வினவல்கள் என்ற வரம்பு இருக்கும். அதைத் தயவுசெய்து கவனத்தில் கொண்டு, பிறகு மெதுவான முறை அல்லது சிறிது இடைவெளி விட்டு மீண்டும் முயற்சிக்கவும்.
