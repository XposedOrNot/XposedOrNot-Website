    XposedOrNot API விசை மேலாண்மை

    இது உங்கள் தற்போதைய API விசை ஆகும், சரிபார்க்கப்பட்ட டொமைன்கள் தொடர்பான மீறல்களை வினவ இதைப் பயன்படுத்தலாம்.

    அதிக கோரிக்கைகள் வந்துள்ளன. தயவுசெய்து வேகத்தைக் குறைத்து சிறிது நேரம் கழித்து மீண்டும் முயற்சிக்கவும்.
    API மீட்டமைப்பு வெற்றிகரமாக முடிந்தது. புதிய API விசையைப் பயன்படுத்தலாம்.
    உங்கள் கோரிக்கையைச் செயலாக்கும்போது ஒரு பிழை ஏற்பட்டது. தயவுசெய்து சிறிது நேரம் கழித்து மீண்டும் முயற்சிக்கவும்.
    curl -L -X POST -H "x-api-key: [KEY]" -H "Content-Length: 0" https://api.xposedornot.com/v1/domain-breaches/
    மீறல் சரிபார்ப்புகள்

    எங்கள் API தரவு மீறல்களைச் சரிபார்க்க அனுமதிக்கிறது, சரிபார்க்கப்பட்ட அனைத்து டொமைன்களிலும் சாத்தியமான பயனர் தரவு வெளிப்பாடு குறித்த புதுப்பித்த நிலையான பார்வையை வழங்குகிறது.

    தடையற்ற ஒருங்கிணைப்பு

    எங்கள் API, SIEM, SOAR மற்றும் பிற பாதுகாப்பு அமைப்புகளுடன் தடையின்றி ஒருங்கிணைக்கும் வகையில் வடிவமைக்கப்பட்டுள்ளது, இது உங்கள் இணையப் பாதுகாப்பு நிலையை மேம்படுத்துகிறது.

    நிகழ்நேரப் புதுப்பிப்புகள்

    எங்கள் API நிகழ்நேரப் புதுப்பிப்புகளை வழங்குகிறது, உங்கள் சரிபார்க்கப்பட்ட டொமைன்களைப் பாதிக்கக்கூடிய புதிய அல்லது உருவாகும் தரவு மீறல்களுக்கு விரைவாகப் பதிலளிக்க உதவுகிறது.

    உறுதியான பாதுகாப்பு

    இந்த API வலுவான பாதுகாப்பு நடவடிக்கைகளுடன் வடிவமைக்கப்பட்டுள்ளது, மீறல் மேலாண்மைக்கான உறுதியான கருவியை வழங்கும் அதே வேளையில் முக்கியமான தரவின் பாதுகாப்பை உறுதி செய்கிறது.



    மேலும் விவரங்களுக்கு எங்கள் API ஆவணப்படுத்தலை மற்றும் API விளையாட்டுத் தளத்தை படிக்கவும்.
